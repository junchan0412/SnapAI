# SnapAI 1.5.0

SnapAI 1.5.0 是一次工作流能力升级版本,重点把项目记忆、历史知识库、命令面板主入口和动作模板库串成更完整的日常使用闭环。

## 主要更新

- 上下文包正式作为项目记忆入口,可保存项目背景、术语表、写作风格、代码栈偏好等长期上下文。
- 历史记录升级为知识库工作流,支持搜索、收藏、删除、标签、按动作/模型/标签筛选,并可从筛选历史创建上下文包。
- 命令面板继续扩展为主入口,可搜索动作、模型、上下文、历史、设置、工作模式、诊断和动作模板。
- 新增正式动作模板库,内置邮件回复、会议纪要、代码审查、中英双语润色、图片理解等常用模板。
- 动作设置页新增“导入动作库”和“导出动作库”,支持 JSON 动作包分享、迁移和备份。
- 动作库导入/导出会自动清除全局快捷键、供应商绑定和模型覆盖,避免导入后产生快捷键冲突或不可移植配置。
- 命令面板新增“添加动作模板”入口,不用进入设置页也可以快速安装内置动作模板。
- 新增动作模板库逻辑测试,覆盖内置模板、分享包导出、导入清洗、重复名称和重复 id 处理。

## 安装提示

SnapAI 当前没有 Apple Developer ID 公证。首次从 GitHub 下载后,如果 macOS 提示应用损坏或无法打开,请先移动到固定位置,再执行:

```bash
xattr -cr /Applications/SnapAI.app
open /Applications/SnapAI.app
```

如果放在用户应用目录:

```bash
xattr -cr ~/Applications/SnapAI.app
open ~/Applications/SnapAI.app
```

应用内更新会尽量自动清理 quarantine,但首次手动下载仍需要这一步。
